# Habit Tracker

Console-Based CRUD application to track habit consistency

## Description

This is a simple console-based application that helps build and maintain your habits. It allows you to Create, Read, Update and Delete your daily habits while tracking your consistency over time.

## Features

- Add new records to habits
- Add new habit types
- View all habits
- Update existing habits
- Delete unwanted habits
